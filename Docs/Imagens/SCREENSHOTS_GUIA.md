## Dicas

- **Tema Claro**: Windows → Configurações → Personalização → Cores → Modo claro
- **Tema Escuro**: Windows → Configurações → Personalização → Cores → Modo escuro
- Tamanho ideal: **800–1200px** de largura
- Formato: **PNG** (para nitidez de texto)
